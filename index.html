உங்கள் data backup & restore பண்ணுங்க
📤 Export — Backup எடுங்க
📂 Import — Data திரும்ப போடுங்க
⚠️ Import பண்ணா இப்போ இருக்கற data replace ஆகும்
📋 எப்படி use பண்றது:
Method 1 — WhatsApp (Easiest):
1️⃣ WhatsApp Share button click
2️⃣ உங்களுக்கே "Saved Messages"-ல் send
3️⃣ New phone-ல் அந்த text copy
4️⃣ "Paste பண்ணு" button click → Done ✅

Method 2 — Google Drive:
1️⃣ Drive button click → link copy ஆகும்
2️⃣ Drive-ல் New Google Doc → paste
3️⃣ New phone-ல் copy → Paste button ✅
